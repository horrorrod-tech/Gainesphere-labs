# Gainesphere-labs
Code and build tools behind Gainesphere — retro game dev, STL/3D pipelines, and other creative-tech side projects.

## Projects

- [`LaunchBox-Hypseus-Singe-Repair-Guide/`](./LaunchBox-Hypseus-Singe-Repair-Guide/) — a written guide and a couple of reference scripts for the genuinely confusing part of running laserdisc/Singe-engine arcade games (including 2-player and lightgun titles) natively in LaunchBox on Windows: figuring out the right folder structure and how each game actually gets linked in. Batocera handles this automatically, but even there it's easy to get lost on which version/folder structure a given game expects when you're the one adding it. Not an app or installer — see that folder's README.
